# Производственная практика в организации "Красный якорь"

## Описание
Данный репозиторий содержит результаты учебной практики по профессиональному модулю **ПМ.04 «Сопровождение и обслуживание программного обеспечения компьютерных систем»**.

## Выполненные задачи
В ходе прохождения практики были успешно реализованы и протестированы следующие функциональные требования:

* **Развертывание инфраструктуры:** Произведена установка и полная настройка локальной среды (XAMPP), включая веб-сервер Apache и СУБД MySQL/MariaDB.
* **Администрирование безопасности:** Настроено дискреционное управление доступом к критически важным файловым ресурсам системы.
* **Автоматизация процессов:** Реализованы и внедрены скрипты автоматизированного резервного копирования данных, минимизирующие риски их потери.
* **Мониторинг и контроль:** Настроена система отслеживания состояния сервера и обеспечения целостности конфигурационных файлов.
* **Документирование:** Разработано руководство оператора в строгом соответствии с требованиями ГОСТ 19.505-79, описывающее все этапы эксплуатации программного комплекса.

Все поставленные в рамках практики задания выполнены в полном объеме, система готова к эксплуатации и дальнейшему масштабированию.
